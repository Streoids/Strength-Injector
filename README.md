# Strength Injector
Minecraft Bedrock edition DLL injector that opens the game alongside

#### And obviously Windows will flag this as a virus but its not, Source code is provided!

## Why Strength:
- Automaticaly Launches the game and injects the DLL.
- Added a check box, when enabled it closes the Injector after injecting the DLL.

## Config:
-config is saved as strengthconfig.txt in the same location as Strength.exe.
-deleting it makes the DLL path reset.
